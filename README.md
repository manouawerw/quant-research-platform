# Quant Research Platform

## Vision

Build an AI-powered institutional-quality stock research platform that:

- Scans the top 500 companies
- Uses official SEC filings and financial reports
- Performs statistical analysis
- Reviews technical charts
- Tracks historical predictions
- Sends iPhone push notifications
- Uses Claude and OpenAI for reasoning
- Helps identify high-conviction opportunities
